Collapse directive for sphinx-bootstrap-theme
=============================================

Installation
------------

.. code-block:: console

    $ pip install sphinx-bootstrap-theme-collapse

Code
----
::

    .. collapse:: Open by default
       :open:

       But closes when you click on it.

    .. collapse:: Closed by default

       But opens when you click on it.

Output
------

.. collapse:: Open by default
   :open:

   But closes when you click on it.

.. collapse:: Closed by default

   But opens when you click on it.
